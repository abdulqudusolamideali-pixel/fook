Learn HTML Images, Maps, Background Images & Picture Elements

1. Standard Image (<img>) Example

The <img> tag displays an image. Always include an alt attribute for accessibility and SEO.

Key Points:
  • Use src attribute for image path
  • Use alt attribute to describe the image
  • Use CSS to make images responsive
  • Images load when needed (lazy loading available)

2. Image Map (<map> & <area>) Example

Click different areas of the stadium to learn about sections. An image map creates clickable regions within an image.

Image Map Explained:
  • <map name="name"> - Container for clickable areas
  • <area> - Defines clickable regions
  • shape - rect, circle, or poly (polygon)
  • coords - Pixel coordinates of the region
  • href - Where the link goes

3. Background Image (CSS) Example

Background images are applied using CSS background-image property. They're used for decorative elements and can be combined with other background properties.

Background Image Properties:
  • background-image: url() - Sets the image
  • background-size: cover/contain - How the image fills space
  • background-position: center - Where the image aligns
  • background-attachment: fixed - Parallax effect
  • background-repeat: no-repeat - Controls repetition

Notice: The hero section at the top uses a background image with a dark overlay for readability. This is a common web design pattern!

4. Picture Element (<picture>) Example

The <picture> element provides responsive images by letting the browser choose the best image for different screen sizes and formats.

Picture Element Benefits:
  • Responsive images for different devices
  • Bandwidth optimization (serve smaller images to mobile)
  • Support for different image formats (WebP, JPEG, PNG)
  • Better performance and user experience
  • Browser chooses best image automatically

Summary: Image Tags Used in This Project

Element Purpose Use Case

<map> & <area> Create interactive hotspots Maps, diagrams, infographics
